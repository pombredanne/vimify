vimify

vim-powered code highlighting for node.js

what?

Let vim highlight your code for the web.

how?

vimify = require 'vimify'
vimify 'x = (y) -> y', 'coffee', (err, style, html) ->
  console.log 'the css', style
  console.log 'the markup', html

where?

npm install vimify
