vimify

vim-powered code highlighting for node.js

what?

vim + code → html + css

how?

vimify = require 'vimify'
vimify 'x = (y) -> y', 'coffee', (err, style, html) ->
  console.log 'the style', style
  console.log 'the markup', html

where?

npm install vimify

or

git clone git@github.com:strathausen/vimify.git
